# Mirai Nihongo

Mirai Nihongo is a **web-first Japanese learning app** focused on helping true beginners move from zero to a clean Pre-N5 foundation, then continue into N5 with clear structure.

## Current direction

This repository is intentionally focused on the **web app** route.

Main product layers:

- `webapp/` — Next.js frontend
- `backend/` — Go API backend
- `content/` — learning content files
- `docs/` — product and technical specifications

## Product goal

Help a beginner move from:

- not knowing where to start
- struggling with kana
- lacking a clear first study path

into:

- completing a solid Pre-N5 foundation
- entering N5 with structure
- seeing progress clearly

## Current priorities

1. build a clean web app structure
2. expose content through backend APIs
3. make Pre-N5 a real interactive learning experience
4. connect dashboard, kana, quiz, and progress to real data

## Repository status

This repository is the clean restart of the project after narrowing the direction to a web-first product.
