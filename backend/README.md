# Backend

This folder contains the Go API for Mirai Nihongo.

## Initial goal

Serve real Pre-N5 content data from the repository so the web app can consume content through stable endpoints.

## First API target

- `GET /health`
- `GET /api/content/pre-n5/manifest`
- `GET /api/content/pre-n5/lessons`
- `GET /api/content/pre-n5/kana`
- `GET /api/content/pre-n5/vocab`
- `GET /api/content/pre-n5/quizzes`
