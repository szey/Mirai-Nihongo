# Start Here: Web App Direction

This project is now focused on the **web app** direction.

## Current product direction

Mirai Nihongo is now being developed as a:

**web-first Japanese learning app for true beginners**

The priority is no longer hardware prototyping.

## What matters now

The active project areas are:

- `webapp/` — Next.js application scaffold
- `backend/` — Go API backend
- `content/` — learning content files
- `docs/` — product, content, and technical specifications

## Main product goal

Build a clean web app that helps a beginner move from:

- not knowing where to start
- not being comfortable with kana
- not having a simple progression path

into:

- confident Pre-N5 completion
- structured movement into N5
- visible progress and repeatable review

## Immediate priorities

1. wire the web app to the Go backend
2. expose content through stable backend APIs
3. turn Pre-N5 into a real beginner experience
4. make dashboard, kana, quiz, and progress pages use real data
