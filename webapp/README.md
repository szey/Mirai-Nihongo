# Webapp

This folder contains the Next.js frontend for Mirai Nihongo.

## Goal

The web app should guide a beginner from zero into a clean Pre-N5 foundation with clear next steps, visible progress, and real content-driven learning.

## Planned core pages

- dashboard
- kana lab
- lesson flow
- quiz
- progress

## Current direction

The frontend should consume real content data from the backend rather than hardcoded lesson data.
